# taylorize
uses taylor series to converge polynomials into an original function
- general taylor series:

  $$\sum_{n=0}^{\infty}\frac{f^{(n)}(a)}{n!}(x-a)^n$$
- i found some patterns related to the polynomial coefficients when differentiating, they were used to write `dx()`.
- `main()` shows how can taylor seems like the original curves in every point.
